# Image Background Remover - MVP 需求文档

## 📋 项目概述

| 项目 | 内容 |
|------|------|
| **项目名称** | Image Background Remover |
| **项目类型** | Web 应用 (SAAS) |
| **核心功能** | 一键去除图片背景 |
| **目标用户** | 设计师、电商卖家、内容创作者 |
| **部署平台** | Cloudflare Pages |

---

## 🎯 MVP 目标

用最少的开发成本，验证核心功能：**用户上传图片 → 自动去除背景 → 下载结果**

---

## 📌 功能需求

### F1: 图片上传
- [ ] 支持点击选择文件
- [ ] 支持拖拽上传
- [ ] 支持格式：JPG, PNG, WebP
- [ ] 文件大小限制：10MB

### F2: 去除背景
- [ ] 调用 Remove.bg API
- [ ] 处理中显示 Loading 状态
- [ ] 错误提示（API 失败、文件过大等）

### F3: 结果展示
- [ ] 原图预览
- [ ] 去除背景后的结果预览
- [ ] 棋盘格背景显示透明效果

### F4: 下载功能
- [ ] 一键下载 PNG（透明背景）
- [ ] 文件名默认为 `removed-bg.png`

---

## 🔧 技术方案

| 层级 | 技术选型 |
|------|----------|
| 前端 | HTML + CSS + Vanilla JS |
| 后端 | Cloudflare Workers (Serverless) |
| API | Remove.bg v1.0 |
| 部署 | Cloudflare Pages (免费) |

### API 调用流程
```
用户上传图片 → 前端转 Base64 → POST /api/remove-bg 
→ Cloudflare Worker 调用 Remove.bg 
→ 返回 Base64 结果 → 前端展示/下载
```
